# **_Medical Store Management System_**
### A **PF-Project** by **Shaheer Ismail** & **Devish Kumar**

---

## **_Overview_**
The **Medical Store Management System** is a Programming Fundamentals (PF) project demonstrating:

-  **File Handling**  
-  **Functions**  
-  **Arrays**  
-  **Pointers**

---

##  **_Purpose_**
This system simulates key operations of a medical store using basic C/C++ concepts such as data storage, retrieval, and processing.

---

##  **_Features_**
-  Add / update / delete medicine records  
-  Manage inventory  
-  Search medicines  
-  File-based data storage
